# إصلاح Mountain Entry — اللغة والإدخال والثيم

تم إصلاح الأسباب الثلاثة:

1. الصفحة كانت عربية فقط لأن النصوص كانت مكتوبة بالعربية ولم تكن تحتوي على طبقة ترجمة. تمت إضافة `mountain-entry-i18n.js` مع دعم العربية والإنجليزية للنصوص الثابتة والديناميكية.
2. زر الإدخال كان موجودًا ولكن قاعدة CSS `.embedded .topbar{display:none}` كانت تخفي شريط الأدوات كاملًا داخل الداشبورد. أصبح وضع الدمج يخفي هوية الصفحة فقط ويُظهر أزرار الإدخال والتحديث والتصدير واللغة والثيم.
3. زر الثيم كان داخل نفس الشريط المخفي. أصبح ظاهرًا ويزامن الوضع الداكن/الفاتح مع Shell الداشبورد.
